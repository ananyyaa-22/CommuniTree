# Product Overview

## CommuniTree Platform

CommuniTree is a dual-track community engagement web application that connects users with their local community through two distinct pathways:

- **Impact Track**: Community service focused on NGO partnerships and volunteering opportunities
- **Grow Track**: Local entertainment focused on hobby-based meetups and social events

## Core Value Proposition

The platform combines social networking with civic engagement, featuring verification systems, safety measures, and gamification to build trust and encourage meaningful community participation.

## Key Features

- **Dual-Track System**: Users can switch between volunteering (Impact) and entertainment (Grow) activities
- **Trust Points System**: Gamified reputation system (0-100 scale) that rewards reliable community participation
- **NGO Verification**: Darpan ID validation system for verified non-governmental organizations
- **Safety Rating System**: Color-coded venue safety badges (Green/Yellow/Red) for event locations
- **Universal Chat Support**: Direct messaging between volunteers and NGO coordinators
- **RSVP System**: Event attendance tracking with trust point implications

## Target Users

- Community-minded individuals seeking volunteering opportunities
- People looking for local entertainment and hobby-based meetups
- NGOs and community organizations seeking volunteers
- Event organizers hosting local activities

## Development Stage

Currently in pre-development phase with complete specifications. Ready for implementation as a high-fidelity responsive web application prototype.
